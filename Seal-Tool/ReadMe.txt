Steps to use Seal Tools:
------------------------

1) Go to cmd prompt and navigate to the path of tools.
   Ex: C:\AppEsteem\SealTools\
2) Select the tool "show_seal.exe" and provide the file path as shown in the example below.
   Ex: show_seal.exe C:\SealedFiles\SampleApplication.exe
3) Output with seal state will be displayed on the console.


  Sample O/P is included as Seal_Output_sealed.png.
